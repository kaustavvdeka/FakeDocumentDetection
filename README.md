# Fake Document Detection System using Blockchain

A full-stack decentralized application for detecting and verifying fake documents using **React**, **Node.js**, and **Blockchain Technology**.  
This project helps users upload, verify, and validate documents securely while maintaining transparency and tamper-proof verification using blockchain.

---

## 🚀 Features

- 🔐 Secure document verification
- 📄 Upload and validate documents
- 🧠 Fake document detection logic
- ⛓️ Blockchain-based document hash storage
- 👤 User authentication
- 📊 Real-time verification status
- 🌐 Decentralized and tamper-proof system
- 📁 IPFS/File storage support (optional)

---

## How to test this locally:
Start your Hardhat node: npm run node
Deploy the new contracts: npm run deploy
Start the Backend: npm run backend
Start the Frontend: npm run frontend
Navigate to the "AI Tools" tab on the navigation bar, paste some ChatGPT-generated text, and see it get flagged in real-time!
